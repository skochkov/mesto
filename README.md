### Содержание

* Описание
* Используемые технологии

**Описание**

Учебный проект в котором можно добавлять фотографии и собирать лайки. Так же есть возможность редактировать информацию о пользователе.

**Используемые технологии**

* Адаптивная верстка
* Grid
* Flex
* Выборка DOM-элементов
* Работа по событию
* Основы JS
